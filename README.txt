DEPLOYMENT — GODADDY (Static Hosting)
--------------------------------------
1. Log into GoDaddy cPanel
2. Navigate to File Manager → public_html
3. Upload the entire contents of the completed/ folder
4. Set index.html as the default document
5. Done — no server-side requirements (pure HTML/CSS/JS)
